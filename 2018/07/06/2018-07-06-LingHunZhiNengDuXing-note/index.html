灵魂只能独行 - note

Posted on 2018-07-06


Introduction

灵魂只能独行 - 周国平 - 人民文学出版社 - 2009

Note

第一辑　灵魂的在场

被废黜的国王

以精神的富有坦然于物质的清贫，是出于精神上的高贵和诚实。

灵魂是一个游子

精神生活 - 对理想和精神价值的追求
完美的境界（完美的美和善）是灵魂的归宿

人的高贵在于灵魂

人以内心世界的巨大差异分出高贵和平庸，乃至高贵和卑鄙。
高贵 - 珍惜内在的精神财富甚于外在的物质财富

在黑暗中并肩行走

敬畏他人的精神本质 - 尊重他人的精神世界的神秘性 ，并给予爱和信任（而不是以理解别人为名义强迫别人袒露内心的一切，一旦遭到拒绝，便斥以缺乏信任）

精神生活的哲学

生活的意义和价值 - 占有精神生活
苦难激发人已有的精神活力，使其在精神上收获。

灵魂的在场

灵魂 - 普遍性的精神在个体的人身上的存在，或超越性的精神在人的日常生活中的存在。
最理想的工作是能体现一个人的灵魂的独特倾向的工作。
度闲的质量取决于灵魂所获得的愉悦。
每个人的灵魂教育都只能是自我教育。

不满足的人比满足的猪幸福

易知足于物质世界但绝不满足于仅仅过物质生活的人的快乐更丰富。

人是有灵魂的

灵魂很可能是人的有关生命的自我意识和精神表达。因此，一个有灵魂的人决不会只爱自己的生命，他必定能体悟众生一体、万有同源的道理。

追求

在精神领域中的追求，目标即寓于过程中，无关世俗的成功，无关显示的姿态。
觉醒的灵魂对虚假的生活有敏锐的觉察和强烈的排斥。
真正的享受需要人的想象力给物添加精神价值。
怀念和追求构成我们的精神生活的基本内容。
精神危机源于执拗的精神追求和自我反省的勇气。
精神需求很高的人必须自己为自己创造精神的食物。
力量有限，追求被失败阻止；目标有限，追求被成功阻止。
真正的成熟，是独特个性的形成，真实自我的发现，精神上的结果和丰收。

人类精神生活的整体

人类精神始终在追求某种永恒的价值，这种追求所形成的持久的精神事业和传统超越一切优秀个人的生死和世代延续。它本身就具有一种永恒的价值，甚至是人世间唯一可能和真实的永恒。
人类精神生活的土壤是统一的，并无学科之分。
（只要心性足够优秀，最深刻最高级的）精神生活可超国界。

第二辑　朝圣的心路

与世界建立精神关系

信仰的实质 - 与世界建立精神关系
人的物质活动、认知活动和社会活动仅是与周围环境的关系，而非与世界整体的关系。唯有人的信仰生活指向世界整体。
一个人（不甘心被世俗生活的浪潮推着走）总想为自己的生命确定一个具有恒久价值的目标，他便是一个有信仰生活的人。
作为信仰生活的支点的那个基本信念无须复杂，相反往往是简单的，但必须是真诚的。
精神生活的基础 - 真诚
敬畏生命，可以获得真诚。

朝圣的心路

每个人正是靠自己的孤独的追求加入人类的精神传统的，而只要你的确走在自己的朝圣路上，你其实并不孤独。

人所能及的神圣

人达成神圣的自然之道是肯定肉体及其欲望的合理性，前提是适当限制欲望。
人身上的神性是由动物性升华而来的。
所谓人性，是动物性向神性的升华。
人的神性基于对神圣价值的信仰。

信仰之光

信仰是相信人生中有比自己的生命更重要的东西。
自己的信仰需要通过独立思考来寻求和确立。
思考信仰的问题最需要真诚（认真和诚实）的态度。

勇气证明信仰

《存在的勇气》 - 蒂利希

  • 有明确的宗教信仰并不证明有勇气，相反，有精神追求的勇气却证明了有信仰
  • 存在性焦虑
    • 人是一种有限的存在（自身中始终包含着非存在）
    • 焦虑 - 意识到非存在的威胁时的状态
      • 一是非存在威胁人的本体上的存在，表现为对死亡和命运的焦虑（在古代末期占上风）
      • 二是非存在威胁人的道德上的存在，表现为谴责和罪过的焦虑（在中世纪末期占上风）
      • 三是非存在威胁人在精神上的存在，表现为对无意义和空虚的焦虑（在现代占主导地位）
        存在的勇气就是自我不顾非存在的威胁而仍然肯定自己的存在。
  • 存在本身不包含一丝一毫的非存在
  • 唯有存在本身具备对抗非存在的绝对力量
  • 接受无意义本身就是有意义的行为，这是一种信仰行为 - 信仰是灵魂的状态（而非头脑里的一种行为）

灵魂状态是最原初的信仰现象，一切宗教观念包括上帝观念都是由之派生的，是这个原初现象的辞不达意的自我表达。

尼采与现代人的精神危机

尼采

  • 真诚的人如何在信仰沦丧的时代生活下去
    • 真诚 - 在信仰问题上不苟且，不盲从，也不伪饰
      • 真诚使人陷入悖论
        • 真诚要求对人生追根究底，把生命的意义看得高于生命本身
        • 对根据的无穷追究必然使任何终极根据站不住脚，生命的意义的丧失使生命本身成了问题
    • 新的信仰
      • 艺术拯救人生
        • 日神精神 - 生活在美的外观中，爱生命的过程，而不去追究它的实质和归宿
        • 酒神精神
          • 早期 - 从个人的痛苦和毁灭中获得与宇宙生命本体相融合的悲剧性陶醉
          • 后期 - 从生命的绝对无意义性中获得悲剧性陶醉：人生是幕悲剧，最大的悲剧就在于它的没有终极根据，但生命敢于承担自身的无意义而并不消沉衰落，这正是生命的骄傲！

伦理失去了宗教的根据，可归入广义的功利。现代人实际上只能在艺术和功利二者间进行选择。

  • 艺术的生存方式注重对生命的体验和灵魂愉悦
  • 功利的生存方式注重对物质的占有和官能享乐

艺术为没有信仰的现代人提供了一种真正的精神补偿。
单纯的物质繁荣并不能使人真正幸福。
