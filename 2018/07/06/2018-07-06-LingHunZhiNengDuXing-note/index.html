灵魂只能独行 - note

Posted on 2018-07-06


Introduction

灵魂只能独行 - 周国平 - 人民文学出版社 - 2009

Note

第一辑　灵魂的在场

被废黜的国王

以精神的富有坦然于物质的清贫，是出于精神上的高贵和诚实。

灵魂是一个游子

精神生活 - 对理想和精神价值的追求
完美的境界（完美的美和善）是灵魂的归宿

人的高贵在于灵魂

人以内心世界的巨大差异分出高贵和平庸，乃至高贵和卑鄙。
高贵 - 珍惜内在的精神财富甚于外在的物质财富

在黑暗中并肩行走

敬畏他人的精神本质 - 尊重他人的精神世界的神秘性 ，并给予爱和信任（而不是以理解别人为名义强迫别人袒露内心的一切，一旦遭到拒绝，便斥以缺乏信任）

精神生活的哲学

生活的意义和价值 - 占有精神生活
苦难激发人已有的精神活力，使其在精神上收获。

灵魂的在场

灵魂 - 普遍性的精神在个体的人身上的存在，或超越性的精神在人的日常生活中的存在。
最理想的工作是能体现一个人的灵魂的独特倾向的工作。
度闲的质量取决于灵魂所获得的愉悦。
每个人的灵魂教育都只能是自我教育。

不满足的人比满足的猪幸福

易知足于物质世界但绝不满足于仅仅过物质生活的人的快乐更丰富。

人是有灵魂的

灵魂很可能是人的有关生命的自我意识和精神表达。因此，一个有灵魂的人决不会只爱自己的生命，他必定能体悟众生一体、万有同源的道理。

追求

在精神领域中的追求，目标即寓于过程中，无关世俗的成功，无关显示的姿态。
觉醒的灵魂对虚假的生活有敏锐的觉察和强烈的排斥。
真正的享受需要人的想象力给物添加精神价值。
怀念和追求构成我们的精神生活的基本内容。
精神危机源于执拗的精神追求和自我反省的勇气。
精神需求很高的人必须自己为自己创造精神的食物。
力量有限，追求被失败阻止；目标有限，追求被成功阻止。
真正的成熟，是独特个性的形成，真实自我的发现，精神上的结果和丰收。

人类精神生活的整体

人类精神始终在追求某种永恒的价值，这种追求所形成的持久的精神事业和传统超越一切优秀个人的生死和世代延续。它本身就具有一种永恒的价值，甚至是人世间唯一可能和真实的永恒。
人类精神生活的土壤是统一的，并无学科之分。
（只要心性足够优秀，最深刻最高级的）精神生活可超国界。
