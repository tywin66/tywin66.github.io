      • 感恩和同情是善良的两面

能否知恩检验人是否存有善根。

    • 第一要善，有同情心

哲学与你有缘

探路幸福–《智慧引领幸福》前言

价值观支配人的主观感受（价值观影响幸福的感受的持久度）。

幸福能否数据化值得怀疑（对用来衡量幸福的因素的重要性的评价，所给的分值，归根到底取决于价值观）。

  • 哲学（立足于价值观来探讨幸福问题）对幸福的理解（大致分两派）
    • 快乐主义 - 幸福就是快乐，强调生命本身的自然性质的快乐和精神的快乐
    • 完善主义 - 幸福是精神上或道德上的完善，承认完善也伴随着精神的快乐
      • 共同点 - 重生命，轻功利，重精神，轻物质

人身上最宝贵的价值是生命和精神，倘若生命单纯，精神丰富，则可称幸福。

真正的享受生命是满足生命本身那些自然性质的需要。

  • 在物质生活有保障后，幸福主要取决于精神生活的品质
    • 良好的智力品质 - 以智力活动的兴趣和习惯为基础找到自己真正喜欢做的事，拥有属于自己的事业（这个意义上的成功带来最大幸福感）
    • 良好的情感品质（快乐的源泉） - 自我的充实，内在生活的丰富，爱的体验和能力
    • 良好的灵魂品质（做人的最高幸福） - 善良、高贵的品德，真诚的信仰

什么是幸福？–威廉·施密特《幸福》中译本序

  • 《幸福》 - 威廉·施密德
    • 序言
      • 写作目的 - 提醒人们思考什么是幸福（而不是盲目追求幸福）
        • 追求幸福的狂热
          • 病因 - 关联破裂，意义缺失（由此产生内心空虚和外在冷漠）
    • 幸福观
      • 不依赖运气
      • 肯定痛苦的价值
    • 充实 - 接受人生根本上的矛盾性，立足于感受真实的、完整的人生（由此产生的幸福感必是深刻而持久的）
    • 人生第一要务是寻求意义，而不是幸福（幸福是片段式的，人生不可避免空虚感）
      • 意义就是关联
        • 一类是我们的生活与有限的生命价值和精神价值的关联（比如父母对子女的爱，出于精神动机从事的事业）
        • 另一类是我们的生活与无限的生命价值和精神价值的关联（对人生的超验意义的信仰）
    • 假设提供超验意义的至高境界（无所谓是否确凿存在）存在，与无限的、神性的充实保持关联才可使人生获得真正充实的幸福

充实的反面-忧虑的价值在于把人引向超验意义的寻求（因而成了充实的幸福的一个重要因子）。

对意义问题的探讨是对幸福问题的探讨的必由之路和归宿（幸福的人生就是有意义的人生）。

人生的三个觉醒

  • 人身上有三个最本质的东西

    • 第一，人是一个生命（从而在世界上生活，有种种人生经历）
    • 第二，人是一个有自我意识的独特的生命个体（人是一个自我）
    • 第三，人是精神性的存在（人是一个灵魂）

  • 人生有三个基本的觉醒

    • 生命的觉醒
      • 透过社会堆积物（财富、权力、地位、名声等）去发现自己的自然的生命，牢记自己是一个生命，对它保持敏感并时时满足它的需要
        • 生命的需要（由自然规定，包括与自然和谐相处、健康、安全等，也包括爱情、亲情、家庭等自然情感的满足）平凡而永恒，是单纯的快乐，不可因其它对复杂的快乐的追求而被损害（若损害单纯的快乐，其价值是可疑的）
    • 自我的觉醒
      • 成为你自己（对自己的人生负责，实现自己的价值，做自己人生的主人）
        • 主要困难
          • 懒惰
          • 怯儒
            • 克服 - 认识到自己的人生的唯一性和有限性
        • 成为了自己的两个可靠标志
          • 一、在人生的态度上自己做主（有明确坚定的价值观，处世做人的原则，在俗世中不随波逐流） - 人生中有真信念
          • 二、在事业的选择上自己做主（全身心投入自己真正喜欢做的事情，并感到内在的愉快和充实） - 事业上有真兴趣
    • 灵魂的觉醒
      • （透过肉身自我去发现自己身上的那个和大我相沟通的精神性自我）认清精神性自我是自己的本质
        • 觉醒途径
          • 信仰
          • 智慧 - 自我与宇宙生命本体合一的境界
          • 共同点 - 摆脱肉身的限制，超越小我，让精神性自我觉醒

  • 精神性自我

    • 觉醒对现实人生的意义
      • 第一，通过做事而做人（灵魂修炼）
      • 第二，具有超脱的心态（在精神上尽量不受无常的人间祸福得失的支配）
    • 不觉醒的话
      • 第一，境界低俗（人沉湎在肉身生活中）
      • 第二，苦海无边（受肉身遭遇支配）

信仰和智慧给人一个超越的立足点。

第四辑 传承高贵

我心目中的好教师

  • 一个好教师该坚持什么
    • 一、智情双修，德才兼备，做一个优秀的知识分子
      • 一个教师的精神素质直接体现在教学过程中
        • 精神素质
          • 智力
          • 情感
          • 道德
      • 和传授知识相比，教师作为一个人在精神上对学生的影响更重要
    • 二、爱学生，真正把学生当作目的
      • （孩子尚缺乏理性判断和情感自主能力）教师的态度直接影响学生对生活和学习的信心
      • 爱学生，就会对学生的成长负责
    • 三、懂教育，拥有正确的教育理念
      • 教育理念与教师的人生观、价值观有密切联系，是教师的整体精神素质在教学上的体现
      • 自觉、坚定地拥有正确的教育理念（而不能只凭直觉）
    • 四、讲究教学艺术，让学生感受到知识的魅力
      • 使学生产生兴趣
        • 兴趣是学习的前提
        • 一个学生对某一门课能否发生兴趣，取决于两个因素
          • 一是学生的天赋类型
          • 二是教师的教学水平
        • 如何做到
          • 首先，对自己所任的课程（教学大纲范围内）在基本原理方面做到融会贯通，能够举一反三
          • 其次，有广泛的知识兴趣和人文修养
    • 五、处理好素质教育和应试教育的关系
      • 一方面帮助学生用最少的时间、最有效的方法对付应试，另一方面最大限度地拓展素质教育的空间
    • 六、淡泊名利，甘于受冷落
      • 一切凭良心做事的人都有一个信念：良心的评判高于体制的评判

剩下的才是教育

    • 一、快乐学习的能力
    • 二、自主学习的能力

  • 教育的目标

    • 第一要让学生喜欢学习，对知识充满兴趣
    • 第二要让学生善于学习，在知识面前拥有自由

知识分子 - 热爱智力生活的智力活动几乎成了本能的人

（怀特海如此划分智力发展的阶段：）小学是浪漫阶段，中学是精确阶段，大学是综合运用阶段；小学和大学都自由，中学则必须是自由从属于纪律。

无论从年龄的特征来说，还是从教学的顺序来说，中学都是最适合于奠定文理知识基础的阶段，知识的灌输最为密集。

法治社会与公民幸福

一切具体的行为，包括对财富的追求，都不是终极目的，而只是实现幸福的手段，其价值都要根据对幸福的贡献得到评定。

不论社会环境怎样，个人在价值观上总能拥有相当的自主权。

  • （法治）社会对个人的幸福承担的责任有两个方面
    • 一是价值观的导向。倘若社会以财富为最高目标，就会形成一种总体氛围，在这种氛围的诱惑和压力下，多数成员在价值选择上必定迷离失措。
    • 二是体制的保障。一个社会唯有能够提供一种制度环境，有助于多数成员争取真正属人的幸福，在生命和精神两方面处于好的状态，才是一个好的社会。

市场经济与法治社会互相依存，同步发展，这既是历史的事实，也是逻辑的必然。和计划经济相配套的是人治，即长官意志，和市场经济相配套的只能是法治，即以保护个人自由为基本原则的法律秩序。从计划经济向市场经济的经济转型若要成功，必有赖于从人治向法治的社会秩序转型也获成功。

  • 从理论上说，法治社会的出发点，就是要寻求一种能够最大限度地保障每个人追求幸福的权利的社会秩序，而这样的秩序可以归结为两个原则
    • 第一个原则是个人自由，即每个人都拥有追求自己心目中的幸福——包括物质利益和精神价值——的权利，只要不损害他人的利益，任何人包括政府不得对其实施强制。 - 保护利己
    • 由此派生出第二个原则，即规则下的自由。 - 惩罚损人
      • 规则的核心则是每个人必须尊重他人的同等权利，如果发生损害他人利益的行为，就要受到强制和惩罚。

市场经济与法治社会密不可分，它无非就是体现在经济领域的法治秩序罢了，是个人享有规则下的经济自由。规则分两类：在私人领域，个人的财产权、公平竞争权等受法律保护；在公共领域，个人须承担由法律规定的包括合理纳税和维护公共利益等义务。法律应当在保护私人利益和维护公共利益方面形成比较完善的体系，政府的责任则是切实执行相关的法律。

  • 幸福是人真正拥有和享受属人的价值，过上了高于动物界的真正人的生活。（两层含义）
    • 在低层次上，是生存获得了基本的物质保障，这本身不是幸福，然而是幸福的前提，不得不为生存挣扎的人仍然生活在动物的境遇中，绝无人的幸福可言。
    • 在高层次上，是对良好的生活品质和精神品质的追求。

法治社会的根本原则是保护个人追求幸福的自由，防止强制的发生，这个原则是用法律来确定和体现的，而法律又是由政府来执行的，因此产生了政府掌握强制权力之必要。

在法律比较完备也就是在整体上确实体现了保护个人自由原则的前提下，法治的重点就在于对政府的强制权力进行有效的限制和约束，以防止其侵犯个人自由。

政府是为人们追求幸福创造良好的制度环境，还是声称要直接为人们创造幸福，这个区别极其重要，法治和人治的分界线就在这里。在法治社会，公民拥有追求幸福的权利，而政府对这个权利加以保护。在人治社会，政府掌握着提供幸福的权力，解释何为幸福的权力，正因为此，也就掌握着剥夺幸福的权力。

唯有推进政治体制改革（脱离向老百姓施予幸福的人治的思路），使政府回归法治社会中应该扮演的角色，才是治本之策。

（临终关怀医院的首倡者希思黎·萨德斯：）只有当我们不再把死亡当作禁忌，我们才能建立起一种与自己的死亡之间的人性的关系。

在天使的微笑之中，有爱，有善良，其实也有艰辛和眼泪。

医患共识：医学的局限性

  • 从医生这方面来说，不可有意无意地给病患制造医学万能的错觉，助长此种幻想

    • 原因
      • 对病患及家属的同情
      • 职业的虚荣心
      • （最应警惕的是）获利的动机（肆意扩张医学的边界）
        • 具体表现
          • 一、过度诊查和过度治疗，造成医源性伤害。相当数量的癌症病人很可能是提前死于化疗之类所造成的医源性伤害。事实上，和一般家庭相比，医生及家属服药最少，在疾病不可治的情形下也最能明智地放弃过度治疗。
          • 二、把生活医学化，正常的生理过程例如生、老、死、女人的怀孕和分娩皆被视为病，被置于医生的控制之下。
          • 三、在诊治上独断专行，病患完全没有发言权。治病是病患最切身之事，有权了解可供选择的不同方案并表达意见，如果不可救治，有权了解实情并做好精神准备。
            • 这种精神状态导致相当一些病患和家属宁愿高估医学的能力，随之而来的是医治失败后对医生的苛责和愤恨。

  • 病患（要认识到医学的局限性）抱有医学万能的幻想

    • 一是因为对医学无知
    • 二是出于对死亡的恐惧
      • 中国人缺乏宗教信仰，有很深的生死迷惑，即使大限临头仍盲目拒绝死亡

医学无论多么进步，始终存在不能治愈只能缓解的疾病，而疾病无论可治不可治，安慰是永远需要的。安慰，即医患之间人性化的交流和合作关系，能够使可治之病的治疗效果更好，也使不治之病人获得最后的尊严。

第十辑　编外三篇

悲智具足南普陀

现代人类通同集注于物质的经济财产方面，贫者怨愤不平，富者穷奢极欲，这条路已走穷，出路在由物质而进到精神，当在精神上求得身命之安全。

人生的伟大真理是相通的，以智慧寻求心灵的觉悟，以慈悲解救人间的苦难，是人类的永恒追求。

孔子曰：“知者乐水，仁者乐山。” —— 知，就是智慧；仁，就是慈悲。智慧，如水一样空灵；慈悲，如山一样坚实。

张晓刚的艺术与现代性中的记忆问题

艺术是心灵的需要，是自救的行为，是要给生活一个意义，因此是超越于世俗的成败的。

所谓艺术气质，就是内省式记忆占主导地位（因此常忽略当下行动，宁愿让自己沉浸在回忆和遐想中）。

内在体验的品质规定了一个艺术家在创作上的可能性。

一个艺术家讨厌浪漫主义，常常是成熟的征兆。

在消费时代，人们以消费主义的态度对待记忆，不求其真实，只问其是否有用。（张晓刚：）记忆成了一个沉重的“包袱”，失忆成了一种“生存本能”和“能力”。

  • 柏格森对两种记忆的区分
    • 习惯性记忆 - 急功近利的时代的人们最需要的
      • 优点 - 对环境迅速做出反应、采取行动
      • 缺点 - 只让对当下行动有用的已经固化为某种习惯的记忆进入意识并发挥作用
      • 实质 - 失忆（只是一种条件反射，用人是精神性存在的尺度衡量）
    • 内省式记忆
      • 优点 - 注重内在体验，在独处中把生活经验整合为一个有意义的整体
      • 缺点 - 延缓甚至不做反应，对当下行动当然是一种妨碍
    • 当社会注重功利的追求时，习惯式记忆排挤内省式记忆乃是必然的事

正视困境本身就意味着一种精神自由。

纪事和感想

人的优点和弱点是相连的，注重内在生活的另一面，便是对外部世界情态的无知。

一个人如何对待持有不同观点的人，比他自己持有何种观点更能反映他的文明程度。舆论的不宽容，对不同观点施以谩骂、语言暴力、道德审判，这种方式所体现的国民素质恰恰是专制的肥沃土壤。个人独裁和多数人的暴政是专制的两极，而两极相通，二者距离民主同样遥远。相对而言，推翻个人独裁还容易一些，铲除多数人暴政的土壤要难得多，这正是我的忧虑。
